# CADWEB - Portafolio de Ingeniería & Desarrollo

Bienvenido al repositorio oficial de **CADWEB**, una plataforma que integra soluciones de **Ingeniería Industrial**, **Diseño Técnico (AutoCAD)** y **Desarrollo de Software Moderno**.

---

## Stack Tecnológico

### Frontend & Mobile
* **Angular 18+** (Standalone Components)
* **Ionic Framework** (Hybrid Mobile Development)
* **TypeScript / JavaScript (ES6+)**
* **SASS / SCSS**

### Backend & Cloud
* **Firebase** (Hosting, Firestore NoSQL)
* **Google Analytics**

### Diseño & Ingeniería
* **AutoCAD** (Modelado 2D/3D y Planimetría)
* **Blender** (Visualización Arquitectónica)

---

## Proyectos Destacados

### proPan (En Desarrollo)
Aplicación móvil diseñada bajo principios de **Ingeniería Industrial** para la optimización de procesos en la industria panadera.
* **Objetivo:** Minimizar mermas y maximizar la eficiencia productiva.

### Servicios de Ingeniería Digital
Soluciones integrales de dibujo técnico y regularización de proyectos.
